# List 
- 여러개의 값을 담을 수 있다. 
```python
list = [1, 2, 3, 4, 5]
```
- 뒤에서 값 가져오기 
```python 
list[-1] # 뒤에서 첫번째 값, 5
```

## 새로운 값 추가하기
```python
# append 함수 사용
list.append(15) 

# 리스트에 리스트 추가 
list2 = list +[3]
```

## 값 삭제하기 
```python
# del 이용 
del list[0] # index 

# remove 이용 
list.remove(44) # 값
```
