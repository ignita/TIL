## 10998번    
https://www.acmicpc.net/problem/1098

### 내 답  
```  
#include <stdio.h>

int main(void)
{
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a * b);
    return 0;
}
```  

### 다른 사람의 답  
https://www.acmicpc.net/source/1215204  
```
#include <stdio.h>
#include <assert.h>
int main()
{
    int num1 = 0, num2 = 0;
    scanf("%d %d", &num1, &num2);
    assert((num1 > 0 && num1 < 10) && (num2>0 && num2 < 10));
    printf("%d", num1 * num2);
    return 0;
}```  
`assert`함수: 코드에서 발생하지 않아야 하는 일을 명확히 지정해줌.  
assert함수 인자가 false가 되면 에러가 발생하고 프로그램 정지.
